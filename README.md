# 🍔 Online Food Ordering System

## 📌 Description
This is a web-based online food ordering website where users can browse food items, add them to cart, and place orders easily.

## 🚀 Live Website
👉 https://madhureddyvangala28-dotcom.github.io/online-food-ordering-system/

## ✨ Features
- User-friendly interface
- Browse food menu
- Add to cart
- Checkout system
- Multiple pages (login, orders, profile)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html (Home page)
- menu.html (Food menu)
- cart.html (Cart page)
- checkout.html (Order page)
- style.css (Styling)

## ▶️ How to Run
1. Download or clone the project
2. Open index.html in browser
